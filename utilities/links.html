Creating Sensitive Areas on a Canvas

This guide explores different methods for creating touch-sensitive areas on a canvas using Arduino.

Arduino Resources

Before diving in, get familiar with Arduino by visiting the official website: Arduino Software.

Find additional resources and tutorials on the Arduino Playground: Arduino Playground.

Methods for Creating Sensitive Areas

Method 1: Arduino Duemilanove + Capacitive Sensing Library

This method uses the built-in Arduino functionality for capacitive sensing. Learn more about:

  • Arduino Duemilanove Board (retired)
  • Capacitive Sensor Library
  • Capacitive Touch Sensing with Arduino Uno Wifi Rev 2 (forum discussion)
  • CapacitiveSensor Library for Teensy (alternative library)

Method 2: Arduino Duemilanove (hopefully) + TTP223 capacitive sensor

This method utilizes a dedicated TTP223 capacitive sensor module. You can find further information on:

  • Interfacing TTP223 Capacitive Touch Sensor with Arduino
  • Interfacing TTP223 Capacitive Touch Sensor with Arduino (Instructables)
  • How to Interface TTP223B Touch Sensor with Arduino (Play With Circuit)
  • Interfacing TTP223 Capacitive Touch Sensor With Arduino (PlayWithCircuit - Youtube)
  • Interfacing TTP223 Capacitive Touch Sensor with Arduino (BasonTech - Youtube)

Method 3: More complex, not clear yet
